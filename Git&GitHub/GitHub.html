Git - version control system, local tool

GitHub - hosting & collaboration provider, web based tool

Repository - the location where your code history is stored

Branch - "Folder" within the repository containing commits

Commit - Different stages of your code inside a branch

Git commands:

    INITIALIZATION

  • git init - initialize the repository
  • git status - see the status of the repository

    • COMMITS

  • git add . - stage all the files in the project
  • git commit -m "..." - create new commit
  • git log - overview of all commits done in the branch located in
  • git checkout ... - checkout branches or commits
  • git checkout -- . go back to the last commit
  • git reset --hard ... - go to the commit we want to, deleting all others

    • BRANCHES

  • git branch - displays all current Branches in our local Repository
  • git checkout -b ... - checkout the branch
  • git merge ... - merge additional branches with master branch
  • git branch -D ... - delete the defined branch
