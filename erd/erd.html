Cars ERD

ERD Description

Table Car Mark

  • Car Mark Table consists of 4 different car marks and all of their specifically assigned IDs.
  • ID attribute shows the ID number, which a certain car mark has been assigned. CarMarkName row shows different car marks
  • The Primary Key for this table is the ID, which acts as a Foreign Key for the Cars In Garage table

Table Cars In Garage

  • Cars In Garage Table consists of 6 different car models and all of their specifically assigned IDs and additionally their according car marks ID
  • ID attribute shows the ID number which a certain car model has been assigned. CarModelName row shows different car models. CarMarkId shows specific car models car mark id
  • The Primary Key for this table is the ID again. Foreign Key for this table is the CarMarkId, which acts as a Primary Key for the Car Mark Table
